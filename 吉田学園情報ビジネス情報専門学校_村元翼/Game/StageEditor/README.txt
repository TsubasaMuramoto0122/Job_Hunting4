// 操作説明
//===========================================================
 < キーボードとマウス >
 ## 通常モード ##
・ＷＡＳＤキー								：移動
・ＢＡＣＫＳＰＡＣＥキー					：一つ前のオブジェクトを削除する
・マウスホイール							：カメラのズーム
・右クリック長押し＋マウスカーソル移動		：カメラ回転
・左クリック  								：オブジェクト設置
・Ｆ３キー									：セーブ
・Ｆ４キー									：ロード
・Ｆ１キー									：座標入力モードに切り替える
・１キーと２キー							：モデルかメッシュのどちらかのモードに切り替える

## モデル設置時 ##
・ＯキーとＬキー							：移動量の変更
・ＺキーとＣキー							：読み込んでいるモデルの種類を切り替える

## メッシュ設置時 ##
・ＯキーとＬキー							：縦のメッシュ数切り替え
・ＩキーとＫキー							：横のメッシュ数切り替え

## 座標入力モード ##
・０から９までのキー						：座標の入力
・左右矢印キー								：ＸＹＺのどの座標入力かを切り替える
・ＥＮＴＥＲキー							：座標の確定
・Ｆ１キー									：通常モードに切り替える

//===========================================================
// ツール説明
//===========================================================
・指定した位置にモデルとメッシュフィールドを設置し
  セーブをすることで、ステージの情報をテキストファイルに保存するツール
  
・専門学校でのゲーム制作で、ステージ制作に関わる作業が多かったので
  次回以降の制作の作業時間短縮のために開発しました。

//===========================================================
// アピールポイント
//===========================================================
・移動のみではなく、座標を直接入力してオブジェクトを設置可能にするために、
  座標入力モードを追加し、０～９のキーを押して、座標を入力するようにしました。
  一文字入力するごとに桁を繰り上げ最大５桁まで入力できるようにしています。
実装部分：game.cpp(457行目～540行目)

・フィールドを並べて設置する際に隙間ができないように、
  フィールド設置はメッシュのそれぞれＸ、Ｚの大きさずつ座標をずらすようにし、
  位置調整をマス目移動のようにしました。
  メッシュの大きさを変えてもずれることがないようにもしています。  		  
実装部分：メッシュの移動→game.cpp(143行目、160行目～165行目)
	      メッシュのサイズ変更→game.cpp(208行目～225行目)(410行目～423行目)
