//===========================================================
// 操作説明
//===========================================================
 < キーボードとマウス >
・十字キー			：選択肢切り替え
・Ｐキー			：ポーズ
・ＥＮＴＥＲキー  	：決定
・ＷＡＳＤキー		：移動・選択肢切り替え
・マウスカーソル	：視点移動
・マウス左クリック	：爆弾を持つ・爆弾を投げる
・スペースキー		：ジャンプ

 < ゲームパッド >
・十字ボタン		：選択肢切り替え
・ＳＴＡＲＴボタン	：ポーズ・決定
・左スティック		：移動
・右スティック		：視点移動
・Ａボタン			：爆弾を持つ・爆弾を投げる
・Ｂボタン			：ジャンプ

//===========================================================
// ルール説明
//===========================================================
・ステージ上にある爆弾生成エリアから爆弾を取って、
　ステージ中の木箱とゴミ袋に投げて全てのオブジェクトを破壊するゲーム
・爆弾の爆発に爆弾を巻き込むと誘爆することができる

//===========================================================
// アピールポイント
//===========================================================
・内積と外積を使った当たり判定をプレイヤーと床・壁・モデルの判定に
　実装し、当たり判定の計算がしやすくなりました。
　実装部分：model.cpp 464行目～754行目 meshfield.cpp 362行目～564行目

・プレイヤーの回転移動をクォータニオンを使った任意回転軸で回転させ、
　より正確な回転移動ができるようにしました。
　実装部分：player.cpp 356行目～370行目
　
・フォントをテクスチャで絵として描画するクラスを作り、指定したフォントで
　フォントサイズやフォント名を指定した文字列を簡単に出せるようにしました。
　今回はスタート画面のゲーム説明文に使用しています。
  実装部分：TextureFont.cpp StringManager.cpp
