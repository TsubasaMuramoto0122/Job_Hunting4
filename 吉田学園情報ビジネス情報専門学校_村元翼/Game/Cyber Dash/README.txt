//===========================================================
// ルール説明
//===========================================================
・自動で前に進むプレイヤーをジャンプボタンのみで操作して、
　障害物やトゲを回避してゴールをめざすシンプルなゲーム。

・途中に存在する黄色や赤色の円はプレイヤーと重なっている状態でジャンプをすると
　空中にいながらより高くジャンプすることができたり、重力が反転するなどのギミックがある。

・他にも途中にゲートがあり、くぐるとプレイヤーの姿が変化し、押すボタンは変わらないが、
　移動方法が変わる。

//===========================================================
// アピールポイント
//===========================================================
・ClearGaugeクラスで、スタート地点からプレイヤーが進んだ距離を差分を取って割合を計算し、
  パーセントとゲージの溜まる量をUIで表示させ、ステージの進行度をわかるようにしました。
実装部分：ClearGauge.cpp (182行目～213行目)
  
・JumpCircleクラスで、2Dの円同士の当たり判定を取り、
  プレイヤーが範囲内でジャンプするとギミックが発動する処理を作り、
  アクション要素を増やしました。
実装部分：JumpCircle.cpp (121行目～195行目)
    
・Meshクラスを作り、SceneクラスとMeshクラスの二つを継承させた多重継承を実装しました。
実装部分：meshクラス、meshcylinderクラス、meshfieldクラス、meshsphereクラスの全て
